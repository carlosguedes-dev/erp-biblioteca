# 📚 ERP Biblioteca

🔗 **[Acessar o Projeto / Repositório](https://github.com/carlosguedes0007-maker/erp-biblioteca)**

## 📌 Sobre o Projeto
Um sistema robusto de gerenciamento de biblioteca (ERP) construído utilizando tecnologias modernas do ecossistema React. O projeto visa facilitar o controle de acervos, empréstimos e o gerenciamento de dados em uma interface rápida e eficiente.

## 🚀 Tecnologias Utilizadas
- **[Next.js 16](https://nextjs.org/)** (App Router)
- **[React 19](https://react.dev/)**
- **[TypeScript](https://www.typescriptlang.org/)**
- **[Prisma ORM](https://www.prisma.io/)** (Gerenciamento de Banco de Dados)
- **[Tailwind CSS v4](https://tailwindcss.com/)** (Estilização Utilitária)
- **Lucide React** (Ícones)

## 🛠️ Como Executar o Projeto

1. Clone o repositório e instale as dependências:
```bash
npm install
```

2. Configure o arquivo `.env` com a URL do seu banco de dados de desenvolvimento.

3. Gere o cliente Prisma:
```bash
npx prisma generate
```

4. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

5. Acesse [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.
